## Gap conhecido

- A tabela `Bonus` ainda não foi convertida integralmente para uma modelagem declarativa dentro de `crafter.json`. O caso necessário para fechar o workbook sample (`Talisman`) foi implementado no engine; os demais efeitos especiais seguem expostos na UI como efeitos herdados, mas sem cobertura numérica completa fora dos cenários já validados.
