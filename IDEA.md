- The main idea is to create a web app game helper, to organize all the data of items, npcs and quests of the games and integrate the information about them, to help the user to get information while gaming with to-do items.

- The first objective is just to organize these text informations and integrate the link between them.

- The second objective is to recreate the crafting and other systems to user simulate uses.

- The third objective is to integrate and save the information that user created in to-do.
